# Design System



This document outlines the core visual style and thematic properties of our design system.



## Theme Overview



Our theme is named Fidelity and is configured for a `dark` color mode, centered around a primary seed color of `#7B3ABF`.



## Typography



The design system utilizes a consistent typographic approach across different elements

 Headlines `inter`

 Body Text `inter`

 Labels `inter`



## Color Palette



The color palette is built upon a foundation with distinct roles for various UI elements

 Primary Color `#7B3ABF` - Used for key interactive elements and branding accents.

 Secondary Color `#5E2C8C` - Provides support for less prominent actions and information.

 Tertiary Color `#9A40C9` - An accent color for highlights or decorative components.

 Neutral Color `#0F0A15` - Serves as a base for backgrounds and non-chromatic elements, ensuring readability and visual hierarchy.



## Shape and Spacing



 Roundedness `2` - Elements feature a moderate level of roundedness, providing a balanced and approachable aesthetic.

 Spacing `2` - A normal level of spacing is applied throughout the UI, ensuring good readability and visual comfort without feeling overly dense or sparse.
